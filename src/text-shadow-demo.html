How to Use:

  • text-shadow - Applies the default text shadow
  • text-shadow-{color} - Applies a text shadow with the specified color (e.g., text-shadow-red-500)
  • text-shadow-[values] - Applies a custom text shadow with arbitrary values (e.g., text-shadow-[2px_2px_5px_white])
  • text-shadow-none - Removes any text shadow
